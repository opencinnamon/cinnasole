### Cinnasole

Cinnasole is a remake of the deleted 'Moonlight Addons' Python script developed by my friend. It was originally built for Windows & Linux, but then Windows mode went legacy, then Linux mode went legacy, then Windows mode was the main developed one, and eventually shut down. Cinnasole is a project built for specifically Linux, and offers ethical-hacking commands like 'nmap', 'dns', 'whois', 'crack', and more.

It supports most (if not, all) Linux distributions such as Ubuntu, Debian, Linux Mint, Arch Linux, Fedora, and more.

---

### Windows

There will never be an *official* Windows port of Cinnasole. You are more than welcome to make one, however just because you are allowed to make one doesn't mean we endorse it. If your computer gets hacked, gets a virus, that is not our fault, it'll be the creator of the Cinnasole Windows port you were using's fault.

> [!NOTE]
> So remember, if someone claims their Windows port of Cinnasole is official, it's not.
> All ports for MacOS or Windows are completely unofficial.

### Guide

To run Cinnasole on your Linux computer, head over to https://console.mtgp.cc/download and download the file.

1. Open your terminal and run this command:
`cd /path/to/cinnasole`

2. Run this command after;
`python3 cinnasole.py`
If you do not have python installed, then head over to [Cinnasole Docs](https://github.com/opencinnamon/cinnasole/blob/main/docs/python-installation.md)
If you need to install Whois, type 'exit' and then type the command to install it.
